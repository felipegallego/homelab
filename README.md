# homelab
My personal homelab
The aim is to have a docker-compose to Manage:
- Music: Navidrome
- Movies and series
    - local: with NAS
    - online: to think 
- Home automation: Home Assistant

## Structure
So the docker containers to manage will be:
- Firewall: controls everything
- Proxmox: control everything
- Navidrome
- Personal NAS
- *To think in one to manage movies/series online*
- Home Assistant
